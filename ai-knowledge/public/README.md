# Deron Public Chatbot Knowledge

Only reviewed, public-safe documents are allowed in this folder.

Allowed content:
- Public website copy that has already been approved.
- Public product descriptions approved for external visitors.
- Public navigation guidance.
- Public support guidance that does not reveal internal systems.

Do not place the following here:
- DMA, APPB, DATCS internal architecture, flight-control implementation, or engineering notes.
- API keys, `.env` files, credentials, tokens, private URLs, or database details.
- Legal drafts, investor documents, pitch decks, unreleased claims, or private technical files.
- Any document that has not been reviewed for public chatbot use.

The public chatbot must only use approved public knowledge from this folder or other explicitly approved public sources.
