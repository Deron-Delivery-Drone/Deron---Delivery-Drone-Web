Hướng dẫn cài đặt

Cài DATCS Desktop sau khi được cấp quyền.

Các bước dưới đây áp dụng sau khi bạn nhận installer đã duyệt hoặc signed download URL từ Deron.

Supported OS
Windows 10/11, macOS 12+, Linux desktop distributions được Deron phê duyệt.
Network
Kết nối internet ổn định để xác thực, đồng bộ cấu hình và kiểm tra quyền truy cập.
Access
Tài khoản Deron/DATCS được cấp quyền. DATCS Web Console là quyền truy cập riêng.
Permission
Một số OS có thể yêu cầu quyền admin để cài đặt.
Windows

Windows

  1. Tải installer Windows đã được phê duyệt.
  2. Mở installer.
  3. Xác nhận publisher/security prompt.
  4. Hoàn tất cài đặt.
  5. Mở DATCS Desktop.
  6. Đăng nhập bằng tài khoản đã được cấp quyền.
macOS

macOS

  1. Tải build macOS đã được phê duyệt.
  2. Mở installer hoặc package.
  3. Nếu macOS cảnh báo bảo mật, dùng quy trình mở ứng dụng tin cậy.
  4. Mở DATCS Desktop.
  5. Đăng nhập bằng tài khoản đã được cấp quyền.
Linux

Linux

  1. Tải package Linux đã được phê duyệt.
  2. Đánh dấu executable nếu package yêu cầu.
  3. Cài bằng package manager hoặc AppImage flow.
  4. Mở DATCS Desktop.
  5. Đăng nhập bằng tài khoản đã được cấp quyền.
Lần chạy đầu tiên

Thiết lập DATCS Desktop trước khi kết nối vận hành thật.

  1. Đăng nhập bằng tài khoản đã được cấp quyền.
  2. Xác minh organization/workspace.
  3. Chọn đúng vai trò vận hành.
  4. Xác nhận device trust.
  5. Đồng bộ cấu hình.
  6. Không kết nối vào live operations nếu chưa được ủy quyền.
Troubleshooting

Các lỗi thường gặp khi truy cập hoặc cài đặt.

Link expired

Yêu cầu link mới. Mỗi signed URL có thời hạn ngắn và không được dùng lại.

Link already used

Mỗi token chỉ dùng một lần. Nếu đã mở hoặc tải trước đó, bạn cần request lại.

Wrong OS

Token được gắn với OS đã phê duyệt. Chọn đúng Windows, macOS hoặc Linux trong request mới.

Installer blocked

Kiểm tra publisher/security prompt. Nếu chữ ký hoặc nguồn không khớp, dừng cài đặt và liên hệ Deron.

Cannot sign in

Xác minh tài khoản đã được cấp quyền và đúng workspace. Không dùng tài khoản Web Console nếu chưa được cấp quyền Desktop.

Web Console vs Desktop

DATCS Web Console chạy tại datcs.deron.vn. DATCS Desktop là app desktop đầy đủ cho workflow desktop, engineering và mission-control.

DATCS Web Console

Bạn muốn vận hành DATCS Web?

DATCS Web Console là môi trường vận hành/truy cập qua trình duyệt dành cho người dùng đã được cấp quyền. DATCS Desktop vẫn là bản đầy đủ cho các workflow desktop, engineering và mission-control.

Bảo mật

DATCS Desktop dùng mô hình tải có kiểm soát, expiry-aware và audit-friendly.

01

Không có public direct download cho installer.

02

Signed URL phải được backend phát hành sau khi token hợp lệ.

03

Token không được lưu vào localStorage và không hiển thị trên UI.

04

Không hiển thị raw backend errors cho người dùng public.

05

Không hardcode private download URLs, version, file size hoặc SHA256 giả.

06

Mỗi request và download approval nên được backend ghi audit log.
